# 台股數據分析專案

## 專案簡介
這是我的第一個數據分析專案，使用Python分析台灣股市數據，包含台積電(2330)和聯發科(2454)的股價趨勢分析。

## 分析內容
- 📈 股價走勢視覺化
- 📊 多檔股票比較
- 💹 成交量分析
- 📉 移動平均線計算

## 使用技術
- Python 3.x
- Jupyter Notebook
- pandas - 數據處理
- yfinance - 股價數據抓取
- matplotlib - 資料視覺化

## 安裝方式

1. 複製這個repository
```bash
git clone https://github.com/你的使用者名稱/taiwan-stock-analysis.git
```

2. 安裝所需套件
```bash
pip install -r requirements.txt
```

3. 開啟Jupyter Notebook
```bash
jupyter notebook
```
